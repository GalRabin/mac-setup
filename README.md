# Setup Mac

2. Install python: `curl https://raw.githubusercontent.com/GalRabin/mac-setup/main/python.sh | bash`
3. Install qwak-sdk: `curl https://raw.githubusercontent.com/GalRabin/mac-setup/main/qwak-sdk.sh | bash`
